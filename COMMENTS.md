# TakeNote


Escolhi o projeto de um editor de notas.
Primeiro passo foi pensar no design que eu queria para a aplicação e depois pensar em como poderia componentizá-la.
Pelo tempo, dividi em 3 componentes individuais (barra de ações, editor, listagem) e 1 componente pai para fazer a junção e comunicação de todos.
Também pensei em 1 componente para pesquisa, mas acabei deixando-o de lado devido ao tempo.
Depois de fazer uma UI mais simples, apenas para dar suporte a criação dos componentes, iniciei a implementar os componentes e criação
de um projeto no firebase e sua integração ao projeto.
Ao terminar os componentes, refinei a UI utilizando Bootstrap e por fim fiz os testes automatizados com protractor.
